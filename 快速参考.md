# MultiGPT 中国版本 - 快速参考指南

## 📍 文件位置速查

```
app/
├── auth/page.tsx           ← 登录/注册（含隐私政策同意）
├── privacy/page.tsx        ← 隐私政策（自动检测语言）
└── terms/page.tsx          ← 服务条款（仅中文）
```

## 🔗 用户访问路由

| 页面 | URL | 说明 |
|------|-----|------|
| 登录/注册 | `/auth` | 中国用户需勾选《隐私政策》和《服务条款》 |
| 隐私政策 | `/privacy` | 根据语言自动显示中文或英文版本 |
| 服务条款 | `/terms` | 中文版本，明确说明不自动续费 |

## ✅ 中国用户流程

### 登录流程
```
输入邮箱密码
    ↓
【必须】勾选 《隐私政策》《服务条款》
    ↓
点击登录
    ↓
验证：userRegion === CHINA && !agreeToPrivacy
    ↓
成功/失败
```

### 注册流程
```
输入邮箱密码确认
    ↓
【必须】勾选 《隐私政策》《服务条款》
    ↓
点击注册
    ↓
验证：userRegion === CHINA && !agreeToPrivacy
    ↓
发送确认邮件/失败
```

## 🛠️ 核心代码

### 验证逻辑
```typescript
if (userRegion === RegionType.CHINA && !agreeToPrivacy) {
  setError("请阅读并同意隐私政策");
  return;
}
```

### 这里有 3 处验证
1. handleSignIn - 密码登录
2. handleOtpSignIn - OTP 登录
3. handleSignUp - 注册

**中国用户登录/注册前必须勾选《隐私政策》和《服务条款》，否则拒绝操作。**
